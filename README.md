使用指南 

host为宿主工程，imageplugin为插件工程

插件apk生成命令

gradlew clean assemblePlugin ,output目录


初次运行项目 最好把app/build目录下的文件全部删除 再clean项目


把插件apk改名Demo.apk放到宿主Assets目录下运行host工程即可
